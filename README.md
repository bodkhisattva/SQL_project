# SQL_project

В данном проекте представлен анализ базы данных с HH с помощью SQL и Python и инструмента, позволяющего писать SQL-запросы прямо в Jupyteer Notebook - psycopg2. В результате анализа были выявлены особенности вакансий, представленных в базе данных, были проведены исследования, обощающие общие тенденции вакансий и предложены варианты будущих исследований, которые могут быть полезны аналитикам рынка труда и соискателям.
